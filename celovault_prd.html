Problem statement
The problem
Users in emerging markets (Nigeria, Kenya, Ghana) hold stablecoins across multiple currencies — USDm, NGNm, KESm, EURm — but have no autonomous way to manage them. Manually monitoring balances, finding the best swap rates across DEXs, and paying in the right local currency requires constant attention. A user in Lagos receiving USDm but needing NGNm for local payments has to manually find rates, switch apps, and execute swaps — often at suboptimal rates and with significant friction.
Solution
CeloVault agent
An autonomous onchain agent that acts as a smart treasury for multi-currency stablecoin wallets. It monitors balances, detects when rebalancing is needed, queries Mento + Uniswap V3 for best rates, executes swaps autonomously, and reports activity to a real-time dashboard. Gas is paid in USDC via fee abstraction — no CELO needed. The agent is registered on ERC-8004 for onchain discoverability and reputation tracking.
Core features
Multi-currency balance tracker
Reads USDm, NGNm, KESm, GHSm, EURm, USDC, USDT balances from a connected wallet on Celo. Displays live balances on dashboard. Polls every 30s.
Autonomous swap engine
Agent queries Mento Protocol and Uniswap V3 (on Celo) for live rates on a configured pair (e.g. USDm → NGNm). Executes swap when user-defined threshold is met. Selects best route automatically. Gas paid in USDC via feeCurrency adapter.
Rate alert system
User sets a target rate. Agent monitors FX rates across DEXs, logs alerts when target is hit, and can be configured to auto-execute on alert.
ERC-8004 onchain identity
Agent is registered in the Celo ERC-8004 Identity Registry as an NFT. Registration JSON (name, description, wallet, endpoints) is hosted and pointed to from the NFT. Reputation feedback submitted after each autonomous swap to build on-chain track record.
MiniPay compatibility
Frontend detects MiniPay browser wallet via window.ethereum.isMiniPay check and auto-connects. Optimised for mobile viewport. Users on Opera Mini with MiniPay see a native-feel experience.
Activity dashboard
Next.js/Tailwind frontend showing: live balances, swap history, FX rate chart (Mento rates over time), agent status (active/paused), transaction log with Celoscan links, and ERC-8004 agent card.

Risks and mitigations
High
Mento SDK unfamiliar — Use Celopedia + docs.celo.org swap examples. Fall back to direct Uniswap V3 calldata if Mento SDK is slow to integrate.
High
7-day timeline is tight — MVP scope is agent + dashboard. MiniPay and multi-chain support are stretch goals. Ship core loop first.
Med
IPFS latency for agent JSON — Use nft.storage with a pinning service. Cache the URI locally. ERC-8004 only needs the URI at registration time.
Med
NGNm liquidity depth on DEXs — Test swap sizes on testnet first. Start with small amounts. Use slippage tolerance of 1-2%.
Low
Railway agent uptime — Use Railway's always-on deployment. Add a health check endpoint on the agent. Keep agent logic stateless and restartable.
