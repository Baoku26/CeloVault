## Key decisions made

### Project name
**CeloVault** — autonomous multi-currency stablecoin wallet agent.

### Primary token pair
**USDm → NGNm** (Nigerian Naira). Chosen because:
- DML is based in Lagos — real local context
- NGNm is live on Celo Mainnet
- Africa-first narrative is strong for hackathon judges
- 14M+ MiniPay users are predominantly in Africa

### Swap routing strategy
Query both Mento Protocol and Uniswap V3 on Celo in parallel.
Return the best rate. Execute on whichever protocol wins.
Mento is expected to win for Mento stablecoin pairs (USDm/NGNm).
Uniswap V3 is the fallback for pairs Mento doesn't support well.

### Fee abstraction
Gas paid entirely in USDC via the adapter address.
Agent wallet must never hold CELO.
USDC adapter: `0x2F25deB3848C207fc8E0c34035B3Ba7fC157602B`

### ERC-8004 approach
Register agent on Celo Mainnet Identity Registry.
Agent JSON hosted on IPFS (nft.storage).
Submit reputation feedback (`giveFeedback`) after every successful autonomous swap.
This keeps the 8004scan ranking climbing throughout the submission window.

### Agent deployment
Railway — always-on Node.js process.
Stateless by design. Safe to restart at any time.
Health check endpoint at `/health`.

### Frontend deployment
Vercel — standard Next.js deployment.

### Design system
Dark mode only. Celo green (`#35D07F`) as primary accent.
shadcn/ui base primitives, heavily customized.
Impeccable commands used iteratively during build.
DESIGN.md generated by `/impeccable document` after design system is established.

### MiniPay detection
```typescript
const isMiniPay = typeof window !== "undefined" && 
  (window.ethereum as any)?.isMiniPay === true;
```
Auto-connect if MiniPay detected. Show optimized mobile layout.
